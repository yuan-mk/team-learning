### Task02：数据读取与数据扩增（3天）

<b>学习目标</b>

### Task03：字符识别模型（3天）
<b>学习目标</b>

<b>字符识别模型</b>

### Task04：模型训练与验证（4天）
<b>学习目标</b>

- 构造验证集
- 模型训练与验证

- 模型调参流程

<b>模型集成</b>
